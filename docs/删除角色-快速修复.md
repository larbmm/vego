# 删除角色 EBUSY 错误 - 快速修复

## 问题
```
删除失败：EBUSY: resource busy or locked, unlink 'memory.db'
```

## 原因
数据库文件被锁定，需要先关闭连接。

## 修复状态
✅ 代码已修复  
✅ 已编译完成  

## 你需要做的

### 重启应用

```bash
# 方法 1：使用 pm2
pm2 restart vego

# 方法 2：直接运行
# 按 Ctrl+C 停止，然后：
npm start
```

### 测试删除

1. 打开 `http://localhost:3000`
2. 进入"角色管理"
3. 删除 xiaoling 角色
4. 应该成功删除，不再报错

## 修复原理

删除前先关闭数据库连接：

```typescript
// 获取角色实例
const char = appInstance.getCharacter(character);
if (char) {
  // 关闭数据库连接，释放文件锁
  char.close();
}
// 然后删除文件
fs.rmSync(workspacePath, { recursive: true, force: true });
```

## 注意

- 删除操作不可恢复
- 会自动备份 config.toml
- 删除后刷新浏览器页面
