## Bot 权限设置

### 推荐权限配置

**不要给 Bot 管理员权限！** 管理员权限会让 Bot 看到所有频道，无法通过频道权限控制。

Bot 需要的权限：
- ✅ 查看频道（View Channels）
- ✅ 发送消息（Send Messages）
- ✅ 读取消息历史（Read Message History）
- ✅ 嵌入链接（Embed Links）
- ✅ 添加反应（Add Reactions）
- ❌ 管理员（Administrator）← **不需要**

### 如何设置

1. 进入服务器设置 → 角色
2. 找到 Bot 的角色
3. 关闭"管理员"权限
4. 开启上述推荐权限

## 私聊设置（私密频道）

### 什么是私密频道？

私密频道是指 **@everyone 角色无法查看** 的频道。Bot 会将私密频道当作"私聊"，每条消息都会回复。

### 创建私聊频道

1. **创建新的文字频道**
   - 右键点击频道分类 → 创建频道
   - 或点击服务器名称 → 创建频道

2. **设置频道权限**
   - 编辑频道 → 权限
   - 找到 @everyone 角色
   - **关闭"查看频道"权限**（设为 ❌）

3. **添加成员权限**
   - 点击"添加成员或角色"
   - 添加你自己，开启"查看频道"
   - 添加 Bot（如"芊芊"），开启"查看频道"

4. **完成**
   - 现在这个频道只有你和 Bot 能看到
   - Bot 会把每条消息都当作私聊回复

### 私聊频道特点

- ✅ 每条消息都会得到回复
- ✅ Bot 会使用更亲密的语气
- ✅ 可以说私密的话
- ✅ 消息不会被其他人看到

### 示例配置

**频道名称**：`芊芊-私聊`

**权限设置**：
```
@everyone     ❌ 查看频道
你自己        ✅ 查看频道
芊芊 (Bot)    ✅ 查看频道
```

## 群聊设置（公开频道）

### 什么是公开频道？

公开频道是指 **@everyone 角色可以查看** 的频道，或者有多个成员的频道。Bot 会将公开频道当作"群聊"，使用智能判断逻辑决定是否回复。

### 创建群聊频道

1. **创建新的文字频道**
   - 保持 @everyone 的"查看频道"权限开启
   - 或添加多个成员

2. **邀请多个 Bot**（可选）
   - 可以邀请多个角色（如芊芊、婉清）
   - 它们会在同一个频道中互动

3. **完成**
   - Bot 会使用群聊判断逻辑
   - 不是每条消息都会回复

### 群聊频道特点

- ⚡ 使用智能判断逻辑（见下方规则）
- ⚡ Bot 会保持得体，避免过于私密的内容
- ⚡ 支持多个 Bot 在同一频道互动
- ⚡ 消息会被其他成员看到

### 示例配置

**频道名称**：`闲聊`

**权限设置**：
```
@everyone     ✅ 查看频道
或
成员A         ✅ 查看频道
成员B         ✅ 查看频道
芊芊 (Bot)    ✅ 查看频道
婉清 (Bot)    ✅ 查看频道
```

## 频道管理技巧

### 删除聊天记录

**方法：删除整个频道**
- 右键点击频道 → 删除频道
- 所有消息都会被删除
- 需要时可以创建同名新频道

### 多角色管理

推荐的频道结构：

```
📁 私聊区
  💬 芊芊-私聊 (私密)
  💬 婉清-私聊 (私密)


📁 群聊区
  👥 三人群聊 (公开)
  👥 测试频道 (公开)
```

### 临时对话

1. 创建临时私密频道"芊芊-临时"
2. 聊一段时间
3. 不需要了直接删除频道
4. 下次需要时再创建新的

### 数据库说明

- 删除频道后，数据库中的记录仍然存在
- 新频道会有新的 `channelId`，Bot 会当作新对话
- 如需彻底清空记忆，在 Web 管理界面删除对话记录

## 群聊响应规则

在公开频道（群聊）中，Bot 使用以下规则判断是否回复：

### 必定回复（100%）

- ✅ @提到 Bot（如 `@芊芊`）
- ✅ 回复 Bot 的消息
- ✅ 消息中包含 Bot 的名字（如"芊芊"）
- ✅ "你们俩" + 命令词（如"你们俩来一下"）

### 高概率回复（75%）

- 🎲 消息包含"你们俩"、"你俩"、"两位"（任何位置）
- 🎲 消息以"你们"、"大家"、"各位"开头

### 中等概率回复（60%）

- 🎲 消息包含问号（？或?）
- 🎲 消息以疑问词开头（怎么、为什么、什么等）

### 低概率回复（20%）

- 🎲 普通消息（不符合以上条件）

### 不会回复（0%）

- ❌ Bot 刚说过话（避免刷屏）
- ❌ 消息太短（少于3个字符）
- ❌ 只有 emoji

### 示例

| 消息内容 | 响应概率 | 说明 |
|---------|---------|------|
| `@芊芊 在吗` | 100% | 提到 Bot |
| `芊芊你在干嘛` | 100% | 包含名字 |
| `你们俩来一下` | 100% | "你们俩" + 命令词 |
| `你们俩怎么样` | 75% | 包含"你们俩" |
| `我觉得你们俩都很好` | 75% | 包含"你们俩" |
| `你们在干嘛` | 75% | "你们"开头 |
| `今天天气怎么样？` | 60% | 包含问号 |
| `为什么会这样` | 60% | 疑问词开头 |
| `今天真开心` | 20% | 普通消息 |

## 常见问题

### Q: Bot 能看到我不想让它看到的频道？

**A:** 检查 Bot 是否有管理员权限。管理员权限会覆盖所有频道权限设置。移除管理员权限后，通过频道权限控制 Bot 的访问。

### Q: 私密频道中 Bot 不回复每条消息？

**A:** 检查频道权限，确保 @everyone 的"查看频道"权限是关闭的（❌）。查看日志确认是否显示 `Channel is PRIVATE`。

### Q: 群聊中 Bot 回复太频繁或太少？

**A:** 可以在配置文件中调整概率：

```toml
[group_chat]
question_response_probability = 0.6  # 问题回复概率（默认60%）
normal_response_probability = 0.2    # 普通消息回复概率（默认20%）
```

### Q: 如何让 Bot 在群聊中更智能地判断？

**A:** 开启 AI 判断模式（会消耗更多 tokens）：

```toml
[group_chat]
use_ai_judgment = true  # 使用 AI 判断是否回复
```

### Q: 删除频道后，Bot 还记得之前的对话吗？

**A:** 不会。新频道有新的 `channelId`，Bot 会当作全新的对话。数据库中的旧记录不会影响新频道。

### Q: 如何彻底清空某个角色的所有记忆？

**A:** 两种方法：
1. Web 管理界面 → 对话管理 → 选择角色 → 删除所有消息
2. 直接删除数据库文件：`.vego/workspace_角色名/memory.db`

### Q: 可以让多个 Bot 在同一个频道互动吗？

**A:** 可以！在公开频道中邀请多个 Bot，它们会：
- 看到彼此的对话
- 根据群聊规则决定是否回复
- 支持多角色互动

### Q: 私聊和群聊的区别是什么？

**A:** 

| 特性 | 私聊（私密频道） | 群聊（公开频道） |
|-----|----------------|----------------|
| 响应方式 | 每条消息都回复 | 智能判断是否回复 |
| 语气 | 更亲密、放松 | 更得体、礼貌 |
| 内容 | 可以说私密的话 | 避免过于私密的内容 |
| 频道权限 | @everyone 不可见 | @everyone 可见 |
