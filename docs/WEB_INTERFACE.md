# Vego Web 管理界面

## 功能特性

### ✅ 已实现

1. **角色管理**
   - 查看所有角色
   - 新增角色（自动创建默认人设文件）
   - 删除角色
   - 编辑角色人设（Markdown 文件）

2. **对话管理**
   - 类似社交软件的聊天气泡界面
   - 查看历史消息
   - 编辑单条消息
   - 多选删除消息
   - 清空所有消息
   - 分页加载

3. **配置管理**（开发中）
4. **系统状态**（开发中）

## 使用方法

### 启动

```bash
npm start
```

程序启动后，Web 界面会自动在 `http://localhost:3000` 启动。

### 访问

在浏览器中打开：
```
http://localhost:3000
```

## 界面说明

### 左侧导航栏

- **👥 角色管理** - 管理所有角色
- **💬 对话管理** - 查看和编辑聊天记录
- **⚙️ 配置管理** - 修改系统配置（开发中）
- **📊 系统状态** - 查看运行状态（开发中）

### 角色管理

#### 新增角色
1. 点击「+ 新增角色」按钮
2. 输入角色名称（只能使用小写字母和下划线）
3. 点击「创建」

系统会自动创建：
- `workspace_{角色名}` 目录
- 默认人设文件（basic.md, personality.md, background.md, speaking.md, rules.md）
- 空的消息文件（messages.jsonl）
- 用户文件（users.json）

#### 编辑角色
1. 点击角色卡片上的「编辑」按钮
2. 选择要编辑的人设文件
3. 在文本框中编辑 Markdown 内容
4. 点击「保存」

#### 删除角色
1. 点击角色卡片上的「删除」按钮
2. 确认删除

⚠️ **注意：删除操作不可恢复！**

### 对话管理

#### 查看消息
1. 在下拉菜单中选择角色
2. 消息会以聊天气泡形式显示
   - 用户消息：绿色气泡，靠右
   - AI 消息：白色气泡，靠左

#### 编辑消息
1. 点击消息气泡右上角的「编辑」按钮
2. 修改内容
3. 点击「保存」

#### 删除消息
1. 勾选要删除的消息（可多选）
2. 点击「删除选中」按钮
3. 确认删除

#### 清空所有消息
1. 点击「清空所有」按钮
2. 确认清空

⚠️ **注意：清空操作不可恢复！**

## 技术栈

- **后端**: Express.js
- **前端**: Alpine.js + Tailwind CSS
- **存储**: JSONL 文件

## 端口配置

默认端口：`3000`

如需修改，编辑 `src/app.ts`：
```typescript
startWebServer(3000); // 改为其他端口
```

## 安全提示

⚠️ Web 界面默认监听 `localhost`，只能本地访问。

如需远程访问，请：
1. 添加身份验证
2. 使用 HTTPS
3. 配置防火墙

## 故障排除

### Web 界面无法访问

1. 检查程序是否正常启动
2. 查看控制台是否有错误信息
3. 确认端口 3000 未被占用

### 角色创建失败

1. 检查角色名称是否符合规范（小写字母和下划线）
2. 确认角色不存在
3. 检查文件系统权限

### 消息加载失败

1. 确认角色已选择
2. 检查 messages.jsonl 文件是否存在
3. 查看浏览器控制台错误信息
