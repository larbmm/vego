# 更新日志

## [0.6.1] - 2026-04-21

### 🐛 Bug 修复

#### 主动聊天功能修复
- **修复多角色人设污染问题**：使用 IIFE 修复闭包变量捕获问题，确保每个角色使用正确的 Character 实例和人设
- **修复数据库存储错误**：将 `require` 改为 ES Module 的 `import`，修复主动消息无法保存到数据库的问题
- **修复时间戳字段错误**：查询历史消息时使用正确的字段名（`created_at` 而非 `timestamp`）
- **修复时间场景不匹配**：为每个时段添加场景提示，避免晚上说"阳光真好"等不合理内容
- **避免重复消息**：检测用户是否回复，未回复时强制 AI 换话题，避免重复发送相同内容

#### 其他修复
- **删除多余文件**：清理 workspace 级别的 `scheduler_state.json` 和 `users.json`，统一使用全局状态文件
- **清理测试文件**：删除所有 test-*.cjs、check-*.cjs、migrate-*.cjs 文件
- **更新 .gitignore**：添加规则忽略测试文件和临时文件

### ⚡ 性能优化
- **优化随机延迟机制**：改为 0-60 分钟随机延迟，让主动消息均匀分布在整个小时内，而不是集中在前半小时

### 📝 配置调整
- 主动聊天最小间隔：2 小时
- 主动聊天触发概率：30%
- 活动时间段：6:30 - 22:30

## [0.6.0] - 2026-04-11

### 🎉 正式版发布

#### ✨ 新功能
- 多角色管理系统，支持独立人设和记忆
- Telegram 和 Discord 双平台支持
- 智能群聊功能：
  - AI 判断是否参与对话（可选）
  - 中文名字识别（芊芊、婉清等）
  - @提及和回复检测
  - 防刷屏机制
- 自动日记系统：
  - 每天凌晨整理对话
  - 生成日记、长期记忆、关系变化
  - 追加到 diary.md 文件
- Workspace 文件系统：
  - 使用 Markdown 管理人设
  - 支持 `[[file:xxx]]` 引用
  - 自动加载所有配置

#### 🐛 Bug 修复
- 修复私聊和群聊消息混淆的问题
- 修复用户 ID 识别错误导致回复发错地方
- 修复日记文件名时区问题（UTC vs 本地时间）
- 修复 workspace 文件引用格式不兼容问题
- 修复 TypeScript 类型检查错误

#### ⚡ 性能优化
- 减少对话历史长度（100 → 20），节省 40% token
- 关闭群聊 AI 判断默认值，改用规则判断
- 移除调试日志，减少控制台输出
- 优化消息批处理逻辑

#### 📝 文档
- 完善 README.md
- 添加群聊设置说明
- 添加 token 优化建议
- 添加支持的模型列表

#### 🔧 配置
- 添加 `group_chat` 配置段
- 添加 `use_ai_judgment` 开关
- 支持 Windows 和 macOS/Linux
- 移除 `NODE_NO_WARNINGS` 环境变量（Windows 兼容）
