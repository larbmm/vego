📖 配置说明

[api] - API 配置

  • key - API 密钥
  • model - 模型名称（如 deepseek-chat）
  • base - API 基础 URL

[character.角色名] - 角色配置

  • path - 工作区路径
  • telegram_bot_token - Telegram Bot Token
  • discord_bot_token - Discord Bot Token

[memory] - 记忆配置

  • max_history_messages - 最大历史消息数
  • compress_threshold - 压缩阈值

[scheduler] - 定时任务配置

  • enabled - 是否启用
  • schedule_time - 执行时间（如 "3:00"）

[proactive_chat] - 主动聊天配置

  • enabled - 是否启用
  • active_hours_start - 活跃时段开始（如 6.5 = 6:30）
  • active_hours_end - 活跃时段结束

[group_chat] - 群聊配置

  • use_ai_judgment - 是否使用 AI 判断
  • question_response_probability - 问题回复概率
