# 更新日志

## [0.6.0] - 2026-04-11

### 🎉 正式版发布

#### ✨ 新功能
- 多角色管理系统，支持独立人设和记忆
- Telegram 和 Discord 双平台支持
- 智能群聊功能：
  - AI 判断是否参与对话（可选）
  - 中文名字识别（芊芊、婉清等）
  - @提及和回复检测
  - 防刷屏机制
- 自动日记系统：
  - 每天凌晨整理对话
  - 生成日记、长期记忆、关系变化
  - 追加到 diary.md 文件
- Workspace 文件系统：
  - 使用 Markdown 管理人设
  - 支持 `[[file:xxx]]` 引用
  - 自动加载所有配置

#### 🐛 Bug 修复
- 修复私聊和群聊消息混淆的问题
- 修复用户 ID 识别错误导致回复发错地方
- 修复日记文件名时区问题（UTC vs 本地时间）
- 修复 workspace 文件引用格式不兼容问题
- 修复 TypeScript 类型检查错误

#### ⚡ 性能优化
- 减少对话历史长度（100 → 20），节省 40% token
- 关闭群聊 AI 判断默认值，改用规则判断
- 移除调试日志，减少控制台输出
- 优化消息批处理逻辑

#### 📝 文档
- 完善 README.md
- 添加群聊设置说明
- 添加 token 优化建议
- 添加支持的模型列表

#### 🔧 配置
- 添加 `group_chat` 配置段
- 添加 `use_ai_judgment` 开关
- 支持 Windows 和 macOS/Linux
- 移除 `NODE_NO_WARNINGS` 环境变量（Windows 兼容）

### 技术栈

- TypeScript 5.3
- better-sqlite3 11.0
- Telegraf 4.15
- Discord.js 14.14
- OpenAI SDK 4.52
